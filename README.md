# CDART Bookstore

A Python console-based bookstore system using MySQL.

## Features
- Search and view all books
- Add books to cart
- Checkout (store pickup or delivery)
- Customer support
- Return books
- Signup/Login system

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set up MySQL database `bookstore`
3. Update DB credentials in `config.py`
4. Run: `python bookstore.py`
